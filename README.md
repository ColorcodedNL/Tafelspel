# Tafelspel
Een speels tafels-oefenspel voor kinderen van groep 3/4.Een speels tafels-oefenspel voor kinderen van groep 3/4. Drie modi: Oefenen (willekeurige sommen met visuele bolletjes en multiple choice), Tafels (doorloop een volledige tafel in volgorde) en Overhoring (toets jezelf met een cijferpad en zie je score). Werkt op iPad en desktop.
